# Changelog

## 1.0.0 — unreleased

Initial release.

- Area / window / fullscreen / all-in-one capture with frozen-screen selection,
  magnifier, dimensions readout and edge snapping
- Scrolling capture with automatic frame stitching (manual + auto-scroll)
- OCR text capture and QR-code recognition (on-device, Vision)
- Screen recording: MP4 (H.264 + AAC) with system audio and microphone
  (macOS 15+), GIF recording, webcam overlay, click highlights, keystroke
  overlay, countdown
- QuickTime-style trim editor with filmstrip, trim handles and playhead;
  export trimmed MP4 or GIF
- Quick Access Overlay with drag-and-drop, hover actions and auto-close
- Annotation editor: arrow, line, shapes, pen, highlighter, text, counter
  badges, blur/pixelate with adjustable intensity, spotlight, crop, and a
  canvas sidebar (padding, backgrounds, rounded corners, shadow)
- Window captures keep their padding/shadow/transparency as editable canvas
  settings instead of baked pixels
- Pin screenshots as floating always-on-top windows
- Capture history, hide desktop icons, configurable global hotkeys
  (⇧⌘3 / ⇧⌘4 / ⇧⌘5 by default)
